# @voyantjs/charters

Opt-in charters module for the Voyant platform. Models luxury yacht-style products that don't fit the cruise schema — per-suite flat pricing, whole-yacht charter under MYBA contract, APA (Advance Provisioning Allowance) as first-class concern.

See [docs/architecture/charters-module.md](../../docs/architecture/charters-module.md) for the full design.

## Status
